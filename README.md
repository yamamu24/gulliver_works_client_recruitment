# react-rails-app_client

## 概要

api を使ったアプリを作れる最小構成のボイラープレートです。
react-rails-app_backend の api と連携させて使うことを想定しています。
公開 api を利用するも良し。

## あらかじめ導入されているもの

- React
- Typescript
- sass
- axios
- react-router-dom
- reset-css

## 使い方

ローカルにひっぱってきたら

```
yarn install
yarn start
```

http://localhost:3500
で立ち上がります。
